# Guillory's Collision Center — Website Redesign Concept

An unsolicited, single-page website redesign concept for **Guillory's Collision Center, Inc.** —
a family-owned auto body and collision repair shop in Gonzales, LA.

## Why this redesign

Their current website (`autocollisionrepairgonzales.com`) currently loads an **error page** — it's
effectively down. Anyone who searches for the shop online finds nothing working, which means missed
estimates and lost trust for a business that has earned a 37-year reputation. This concept shows a
fast, modern, mobile-friendly site that puts the phone number, free-estimate request, and lifetime
warranty front and center.

## What's included

- Sticky, shrinking header with click-to-call `(225) 622-4144`
- Bold hero with the **"Get a Free Estimate"** call to action
- Services overview (collision repair, auto body & paint, frame/structural, insurance claims,
  free estimates, lifetime warranty)
- A real, styled **Free Estimate request form** (name, phone, vehicle year/make/model, damage
  description, optional photo upload) — the shop's money feature
- "Why us" trust section, illustrative customer testimonials, and location/hours/contact
- Smooth scroll-reveal animations, responsive from 360px phones to widescreen, and
  `prefers-reduced-motion` support

## How to view

Just open `index.html` in any browser (double-click it). No build step, no dependencies —
it's fully static.

## Notes

- The estimate form is a front-end demo; it is **not wired to a backend** and sends nothing.
- Imagery uses CSS gradients and inline SVG placeholders (marked with `data-photo`) rather than
  hotlinked photos, so real shop photos can drop in later.
- This is an unsolicited concept pitch, not the official Guillory's Collision Center website.
